# 智能记账 App

一个支持自然语言输入的网页版记账应用，调用智谱 GLM-4-Flash AI 解析记账内容。

## 功能

- 🎤 **自然语言记账**：说句话就能记，比如「今天去陈记顺和吃了200块」
- 🤖 **AI 自动解析**：自动提取金额、分类、描述、日期
- 💾 **本地存储**：数据存在浏览器里，隐私安全
- 📊 **实时统计**：本月支出、分类统计、最近记录

## 使用方法

1. **申请 API Key**
   - 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/)
   - 注册账号，创建 API Key（免费额度足够用）

2. **打开应用**
   - 用浏览器打开 `index.html`
   - 输入 API Key（会保存在本地）

3. **开始记账**
   - 在输入框里说话，比如：
     - 「打车花了35」
     - 「买了杯奶茶18」
     - 「昨天交房租2500」
     - 「周末看电影花了80」
   - 点击「记账」，AI 会自动解析
   - 确认无误后保存

## 技术栈

- HTML5 + Tailwind CSS（界面）
- 原生 JavaScript（逻辑）
- LocalStorage（数据存储）
- 智谱 GLM-4-Flash（AI 解析）

## 注意事项

- API Key 只存在你的浏览器里，不会上传到任何地方
- 数据只存在本地，换浏览器/清缓存会丢失
- 后续可升级云端同步版本
